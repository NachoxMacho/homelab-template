# Talos Upgrade Checklist

The following is a simple checklist to prevent issues when upgrading talos.


1. [ ] New version is determined.
1. [ ] Image is set to install the QEMU agent
1. [ ] Kernel parameters are set properly
1. [ ] Updating only 1 minor version at a time and 1 node at a time
1. [ ] Services in cluster are healthy before upgrade
